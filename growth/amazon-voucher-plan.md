# Amazon / Voucher Channel Plan

This is a planning document, not an instruction to list immediately. Marketplace rules for digital codes, gift cards, vouchers, and API credits can be strict and can change.

## Safer Alternatives First

1. Sell credits on the TKEN site directly with card and USDT payment.
2. Create downloadable onboarding PDFs or setup services on marketplaces where digital services are allowed.
3. Use affiliate or creator codes that redeem on the TKEN site, not Amazon gift-card-style products.
4. Sell a legitimate consulting/setup package if the marketplace permits services.

## Risks To Check Before Amazon Listing

- Whether third-party API credits or redemption codes are allowed.
- Whether digital delivery is allowed for the seller account category.
- Whether "gift card", "voucher", or "recharge code" language triggers restrictions.
- Whether AI/API service credits require additional approval.
- Refund and chargeback exposure for instantly delivered codes.
