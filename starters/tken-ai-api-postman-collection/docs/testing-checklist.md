# Testing Checklist

1. Set `base_url` to `https://www.tken.shop/v1`.
2. Set `api_key` to your TKEN API key.
3. Run `Models`.
4. Run `Chat Completion` with `free-model`.
5. Switch to `premium-gpt` for harder coding or reasoning tests.
6. If a 401 appears, recheck the API key.
7. If a 502 appears, retry and check route availability.
