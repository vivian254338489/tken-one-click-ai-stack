# Railway TKEN Setup

Add these variables in Railway:

| Variable | Value |
| --- | --- |
| `OPENAI_BASE_URL` | `https://www.tken.shop/v1` |
| `OPENAI_API_KEY` | your TKEN API key |
| `DEFAULT_MODEL` | `tken-free-model` |
| `PREMIUM_MODEL` | `premium-gpt-model` |

If the upstream project uses different variable names, map them using `provider-map.md`.
