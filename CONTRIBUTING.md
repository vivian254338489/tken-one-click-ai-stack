# Contributing

Useful contributions:

- Deployment fixes for Railway, Render, Vercel, Docker, or other hosts.
- New OpenAI-compatible client config templates.
- Safer production defaults.
- Better README examples and screenshots.
- Bug reports with exact commands and logs.

## Local Check

```bash
npm install
npm run check:kits
```

## Rules

- Do not commit API keys.
- Keep brand references accurate and non-affiliated.
- Keep examples OpenAI-compatible.
- Keep install packages runnable after unzip.
