# Troubleshooting

## 401 Unauthorized

Check that the key is current and sent as a bearer token.

## 502 Gateway Error

Retry once, then check the selected route and account balance.

## Model Not Found

Start with `free-model` and `premium-gpt`, then replace with your account's route names if needed.

## Wrong Base URL

Use the `/v1` endpoint:

```text
https://www.tken.shop/v1
```
