# Next Steps

1. Double-click `START.bat` on Windows, or run `sh START.sh` on macOS/Linux.
2. Open `http://localhost:8789`.
3. Paste your API key in the UI.
4. Send a test message.
5. If startup or config fails, double-click `SUPPORT.bat` or run `sh SUPPORT.sh`, then send `generated/support-report.json`.

To change endpoint/model defaults:

```bash
npm run configure
```

Use TKEN:

```text
https://www.tken.shop/v1
```
