# TKEN VS Code Continue Config Pack

Config examples for VS Code coding assistants that support OpenAI-compatible API base URLs.

Default API base:

```text
https://www.tken.shop/v1
```

Get an API key:

```text
https://www.tken.shop/?utm_source=github&utm_medium=readme&utm_campaign=tken_vscode_continue_config_pack
```

## Quick Start

1. Open your coding assistant config.
2. Use an OpenAI-compatible provider.
3. Set base URL to `https://www.tken.shop/v1`.
4. Add `free-model` for low-cost work.
5. Add `premium-gpt` for harder coding tasks.

## Included Files

| File | Purpose |
| --- | --- |
| `configs/continue.config.json` | Generic Continue-style config example |
| `configs/model-policy.json` | Low-cost and premium route policy |
