AI API troubleshooting

Fix OpenAI-compatible API errors faster.

Start with auth, base URL, model route, gateway health, and CORS. Most problems are one of these five.

Try TKEN API View examples

401 Unauthorized

Check the Authorization header and key scope.

502 Gateway Error

Check upstream URL, upstream key, and provider availability.

Model Not Found

Use local route names like free-model and premium-gpt.

API Base URL

Use https://www.tken.shop/v1 and avoid duplicated `/v1`.

CORS Error

Set a production-safe `CORS_ORIGIN` or call from your backend.
