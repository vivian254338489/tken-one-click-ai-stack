# TKEN API Cost Calculator

Static calculator for comparing premium-only AI API usage against route-based usage with low-cost and premium model routes.

Open `index.html` in a browser or deploy it to any static host.

Live API base used in examples:

```text
https://www.tken.shop/v1
```

Try TKEN:

```text
https://www.tken.shop/?utm_source=github&utm_medium=readme&utm_campaign=tken_api_cost_calculator
```

## What It Estimates

- Premium-only monthly spend
- Routed monthly spend
- Monthly savings
- Break-even notes for simple vs hard tasks

## Use Cases

- API pricing comparison content
- GPT vs Claude vs low-cost model planning
- Sales conversations with cost-sensitive users
- Internal ROI estimates before switching clients to route-based model use

## Deploy

- Vercel: import this repository as a static site.
- Netlify / Cloudflare Pages: deploy the folder as static files.
- GitHub Pages: serve `index.html`.

## Disclosure

This is a planning calculator, not a provider quote. Actual cost depends on tokens, model route, context length, retries, and provider terms.
