# Vercel TKEN Setup

Add these environment variables in Vercel:

| Variable | Value |
| --- | --- |
| `OPENAI_BASE_URL` | `https://www.tken.shop/v1` |
| `OPENAI_API_KEY` | your TKEN API key |
| `DEFAULT_MODEL` | `tken-free-model` |
| `PREMIUM_MODEL` | `premium-gpt-model` |
