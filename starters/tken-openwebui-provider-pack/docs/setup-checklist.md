# Setup Checklist

1. Create or copy a TKEN API key.
2. Set API base URL to `https://www.tken.shop/v1`.
3. Add `free-model` for low-cost work.
4. Add `premium-gpt` for coding or hard reasoning.
5. Send one short test chat.
6. If the model is not found, verify the route name in your TKEN dashboard.
