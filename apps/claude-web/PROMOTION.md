I made a small Claude-style Web UI for testing OpenAI-compatible endpoints.

It is intentionally simple: no database, no signup flow, just a browser UI that can point at one `/v1` API base URL and switch between low-cost and premium model routes.

Disclosure: I work on TKEN-related tooling. The default config points to a TKEN-compatible endpoint, but you can edit `public/config.js` for any OpenAI-compatible provider.
