# SEO Site Network Plan

Goal: build useful, non-duplicate content sites that route qualified users to `https://www.tken.shop/` and the one-click GitHub kit.

Do not create thin doorway pages. Each site needs unique content, screenshots, examples, and real utility.

## Site 1: GPT Tutorial Site

Positioning:

```text
Practical GPT API tutorials for builders who want lower blended AI API cost.
```

Core pages:

- 3-minute ChatGPT gateway setup
- OpenAI-compatible API examples
- GPT cost reduction with route fallback
- Codex client setup with TKEN `/v1`
- FAQ: API key, base URL, model names, billing, limits

Soft CTA:

```text
Want to test the examples without changing your client? Use the TKEN OpenAI-compatible endpoint.
```

## Shared Rules

- Use canonical links when reusing content from the main site.
- Do not copy the same article across all sites.
- Add screenshots or command examples on every tutorial page.
- Add UTM links per site:
  - `utm_source=gpt_tutorial_site`
  - `utm_source=claude_tutorial_site`
  - `utm_source=ai_tools_nav`
