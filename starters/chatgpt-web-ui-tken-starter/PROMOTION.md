## Short Description

```text
ChatGPT-style Web UI starter for TKEN and OpenAI-compatible APIs. Deploy with Railway, Vercel, Docker, or local Node.js.
```

## Reddit Soft Launch

```text
I made a minimal ChatGPT-style Web UI starter for testing OpenAI-compatible endpoints.

It is intentionally small: no database, no auth system, just a local UI that can point at one `/v1` base URL and switch between low-cost and premium model routes.

Disclosure: I work on TKEN-related tooling. The starter defaults to a TKEN-compatible endpoint, but the config can be changed to any compatible provider.
```
