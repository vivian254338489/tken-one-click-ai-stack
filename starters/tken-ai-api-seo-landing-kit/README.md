# TKEN AI API SEO Landing Kit

Static landing page kit for AI API comparison, FAQ, and tutorial search traffic.

Default API base:

```text
https://www.tken.shop/v1
```

Main site:

```text
https://www.tken.shop/?utm_source=github&utm_medium=readme&utm_campaign=tken_ai_api_seo_landing_kit
```

## Pages

| Page | Search Intent |
| --- | --- |
| `public/index.html` | cheapest AI API and OpenAI-compatible gateway |
| `public/gpt-vs-claude-api.html` | GPT vs Claude API comparison |
| `public/chinese-model-api.html` | Chinese model API for international users |
| `public/faq.html` | API base URL, model routes, billing, and troubleshooting |

## Quick Start

```bash
npm install
npm run check
```

Deploy `public/` to any static host after owner approval.
