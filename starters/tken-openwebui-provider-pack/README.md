# TKEN Open WebUI Provider Pack

Copy-paste OpenAI-compatible provider settings for Open WebUI-style frontends.

Default API base:

```text
https://www.tken.shop/v1
```

Get an API key:

```text
https://www.tken.shop/?utm_source=github&utm_medium=readme&utm_campaign=tken_openwebui_provider_pack
```

## Quick Start

1. Open your WebUI admin settings.
2. Choose an OpenAI-compatible provider.
3. Set API base URL to `https://www.tken.shop/v1`.
4. Paste your TKEN API key.
5. Add model routes such as `free-model` and `premium-gpt`.
