## Short Description

```text

```

## Soft Launch Draft

```text
I made a small set of task-based LLM routing recipes.

The idea: stop sending every request to the most expensive model. Route summaries, extraction, translation, and support drafts to low-cost routes, then reserve premium GPT-style routes for coding, hard reasoning, and final answers.

Disclosure: I work on TKEN-related tooling. The examples use TKEN route names, but the pattern works with any OpenAI-compatible gateway.
```
