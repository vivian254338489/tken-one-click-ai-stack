# TKEN Free Model Fallback Recipes

Routing recipes for sending simple work to low-cost routes and escalating hard work to premium routes.

Default low-cost route: `free-model`
Premium escalation route: `premium-gpt`

Default API base:

```text
https://www.tken.shop/v1
```

Get an API key:

```text
https://www.tken.shop/?utm_source=github&utm_medium=readme&utm_campaign=tken_free_model_fallback_recipes
```

## Quick Start

```bash
npm install
npm run demo
```
